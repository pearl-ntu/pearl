# Setup Instructions: Create Repository on pearl-ntu

## Prerequisites
Make sure you have access to the `pearl-ntu` GitHub account/organization.

## Step 1: Create the Repository on GitHub

1. **Go to GitHub:**
   - Visit: https://github.com/organizations/pearl-ntu/repositories/new
   - Or: https://github.com/new (make sure you're creating it under `pearl-ntu`)

2. **Repository Settings:**
   - **Owner:** Select `pearl-ntu` (from the dropdown)
   - **Repository name:** `pearl` (or your preferred name)
   - **Description:** "Photon Emission and Reactivity Laboratory Website"
   - **Visibility:** ✅ **Public** (required for free GitHub Pages)
   - ⚠️ **DO NOT** check any of these:
     - ❌ Add a README file
     - ❌ Add .gitignore  
     - ❌ Choose a license
   - Click **"Create repository"**

## Step 2: After Repository is Created

Once the repository exists, come back and we'll push the code to it.

The repository should be at: **https://github.com/pearl-ntu/pearl**

## Step 3: Push Your Code

After creating the repository, run:

```bash
git push -u origin main
```

## Step 4: Enable GitHub Pages

1. Go to: https://github.com/pearl-ntu/pearl/settings/pages
2. Under **"Source"**:
   - Select: **"Deploy from a branch"**
   - Branch: **`main`**
   - Folder: **`/ (root)`**
3. Click **"Save"**
4. Your site will be live at: **https://pearl-ntu.github.io/pearl/**
