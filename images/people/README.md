# People Photos Directory

This directory contains photos of team members.

## How to Add Photos:

1. **Prepare your photos:**
   - Recommended size: 400x400 pixels (square)
   - Format: JPG or PNG
   - Good quality headshot photos work best

2. **Naming convention:**
   - Use lowercase letters and hyphens
   - Examples:
     - `syed-ali-abbas-abedi.jpg`
     - `rongrong-huang.jpg`
     - `guanyu-jiang.jpg`
     - `lovelesh.jpg`

3. **Add photos to this directory:**
   - Place the photo files directly in this `images/people/` folder
   - The website will automatically display them

4. **Photo will appear automatically:**
   - Once you add a photo with the correct filename, it will show on the People page
   - If a photo is missing, a placeholder icon will be displayed instead
